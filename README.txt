# CRACK-GROWTH-RETARDATION-MODELS-UNDER-LOAD-INTERACTION
In this repository, you will find all scripts codes developed and used at the work "CRACK GROWTH RETARDATION MODELS UNDER LOAD INTERACTION".
All codes were write at 2020b MatLab version.

The codes replicate the Wheeler, Willenborg, Gallergh models and the modified Newman's Strip Yield model.
